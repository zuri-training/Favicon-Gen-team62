How to add favicon to a website - ICO format

Step 1: Name Your Favicon

Make sure that your favicon is named favicon.ico. The naming of the file is important because web browsers automatically look for a file with that specific name.

Step 2: Upload Your Favicon

Upload your favicon to your website's main or root directory. The file should be accessible at yourwebsite.com/favicon.ico. The location of the file is important because web browsers will automatically look for a file named favicon.ico in that specific location.

How to add favicon to a website - PNG format

Step 1: Name Your Favicon

While not required, we recommend that you name your PNG formatted favicon favicon.png. If you want to use a custom name then head to this advanced tutorial.

Step 2: Upload Your Favicon
